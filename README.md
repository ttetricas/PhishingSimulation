# Phishing Simulation

Projeto acadêmico desenvolvido para demonstrar como ataques de phishing funcionam.

## Funcionamento

- Página falsa de login
- Envio de dados via formulário
- Armazenamento em banco de dados
- Redirecionamento do usuário

## Tecnologias

- HTML
- CSS
- PHP
- MySQL

## Aviso

Este projeto é apenas para fins educacionais. Não deve ser utilizado para atividades maliciosas.
